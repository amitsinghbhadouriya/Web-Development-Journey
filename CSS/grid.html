What is CSS Grid?

CSS Grid Layout is a two-dimensional layout system for the web. It lets you layout items in rows and columns, making complex designs much simpler to implement compared to older methods like floats or inline-block.

Key Properties Used:

  • display: grid; — Turns the container into a grid layout.
  • grid-template-columns: repeat(4, 1fr); — Defines 4 equal-width columns.
  • grid-gap: 9px; — Adds 9px space between grid rows and columns.
  • 1fr — A fraction of the available space in the grid container.

Responsive Grid:

Using @media queries, the number of columns can adapt based on screen size:

  • repeat(2, 1fr) for tablets and medium screens.
  • 1fr (one column) for small screens or phones.
