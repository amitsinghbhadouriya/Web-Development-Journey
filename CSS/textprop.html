Text Properties in CSS

What are text properties?
Text properties in CSS allow you to control the appearance and layout of the text on a webpage. They help improve readability, design consistency, and emphasize certain parts of your content. Common text properties include text-align, text-decoration, font-size, font-style, text-transform, and more.

Types of Text Properties

  • text-align: Aligns text (left, right, center, justify)
  • text-decoration: Underline, overline, or line-through
  • text-transform: Capitalize, uppercase, lowercase
  • line-height: Sets space between lines of text
  • font-family: Defines font style
  • font-size: Controls text size
  • font-style: Normal, italic, or oblique
  • font-variant: Small-caps effect
  • font-weight: Boldness of text
