CSS Animation & Transform Properties

  • transform: Applies changes like scale, rotate, translate.
  • scale(): Enlarges or shrinks an element.
  • translateY(): Moves element vertically.
  • transition: Smooths property changes over time.
  • box-shadow: Adds depth with shadows.
  • animation: Runs predefined keyframe motion.
  • @keyframes: Defines the animation stages.
