1. Static Position (Default)

position: static — This is the default value. Elements appear in normal document flow, top to bottom, left to right.

Static

2. Relative Position

position: relative — Moves the element relative to its original position without affecting other elements.

Relative

3. Absolute Position

position: absolute — Positions the element relative to the nearest positioned ancestor. It is removed from normal flow.

Absolute

4. Fixed Position

position: fixed — Sticks the element to the viewport. It stays in place even when you scroll.

Scroll down to see the fixed box stick to the screen.

5. Sticky Position

position: sticky — Acts like relative until it reaches a specified scroll position, then behaves like fixed.

Sticky
