Understanding CSS Display Property

What is the Display Property?

The display property in CSS determines how an element is visually represented and behaves in the layout flow of the document. It decides whether the element will appear as a block, inline, flex container, grid, or be completely hidden.

Why Use It?

  • To control layout and positioning of elements
  • To align elements horizontally or vertically
  • To hide or show elements dynamically
  • To build responsive and flexible UI layouts

Summary Table

Display Type Description
block Starts on a new line, takes up full width
inline Stays on the same line, width/height cannot be set
inline-block Inline position with block features (width/height)
none Completely hides the element from layout
flex Creates a 1D flexible layout (row or column)
grid Creates a 2D grid layout
inline-flex Same as flex but acts like an inline element
