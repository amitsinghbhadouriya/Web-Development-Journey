CSS Units Demonstration

This box can use em or rem units for font size.
Try uncommenting the CSS to test the difference!
This box uses vw and vh units to take 50% of the screen width and height.

Common CSS Units:

  • px (pixels): Fixed size unit.
  • em: Relative to the font size of the parent element.
  • rem: Relative to the font size of the root (html) element.
  • %: Relative to the size of the parent container.
  • vw (viewport width): 1vw = 1% of the viewport width.
  • vh (viewport height): 1vh = 1% of the viewport height.
