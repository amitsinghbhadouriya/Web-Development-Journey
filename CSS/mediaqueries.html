What is a Media Query?

A media query is a CSS technique used to apply styles conditionally based on the user's device characteristics, like screen width, height, resolution, or orientation.

Example Breakdown:

  • @media screen and (max-width: 800px): Targets screens that are 800px wide or less (like tablets or phones).
  • body { background-color: blue; }: Changes the background color of the page when on smaller screens.
  • .container changes from red/white text to yellow/black text on smaller screens.

Why Use Media Queries?

  • To make websites responsive and mobile-friendly.
  • To optimize layout and readability across devices.
  • To reduce the need for separate mobile websites.
