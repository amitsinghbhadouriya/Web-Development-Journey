Margin Collapse in CSS

What is margin collapse? Margin collapse occurs when the vertical margins of two block-level elements meet. Instead of adding together, only the larger margin is used. For example, if one element has a bottom margin of 30px and the next element has a top margin of 30px, the space between them will be just 30px â€” not 60px.
