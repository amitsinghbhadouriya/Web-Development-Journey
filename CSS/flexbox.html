Flexbox Layout Overview

Flexbox is a layout model in CSS that allows items within a container to be arranged dynamically and flexibly. It helps in creating responsive and adaptable layouts, making it easier to distribute space, align items, and handle wrapping.

1. display: flex

Sets the container to use Flexbox. All direct child elements of the container become flex items.

2. flex-direction

Defines the direction of flex items within the container.

  • row: Default. Items are arranged horizontally (left to right).
  • column: Items are arranged vertically (top to bottom).
  • row-reverse: Items are arranged horizontally in reverse (right to left).
  • column-reverse: Items are arranged vertically in reverse (bottom to top).

3. flex-wrap

Defines whether the flex items should wrap to the next line if there is not enough space.

  • nowrap: Items do not wrap and stay in one line (default).
  • wrap: Items will wrap onto multiple lines if necessary.
  • wrap-reverse: Items wrap in reverse order.

4. justify-content

Aligns flex items along the main axis (default is horizontal).

  • flex-start: Items are aligned at the start of the container (default).
  • flex-end: Items are aligned at the end of the container.
  • center: Items are centered within the container.
  • space-between: Items are evenly distributed with space between them.
  • space-around: Items are evenly distributed with space around them.
  • space-evenly: Items are evenly distributed with equal space between and around them.

5. align-items

Aligns items along the cross axis (default is vertical).

  • flex-start: Items are aligned to the start of the cross axis.
  • flex-end: Items are aligned to the end of the cross axis.
  • center: Items are centered along the cross axis.
  • baseline: Items are aligned based on their baseline.
  • stretch: Items stretch to fill the container (default).
